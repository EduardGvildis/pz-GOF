# Патерн Спостерігач (Observer)

Патерн дозволяє об'єктам (спостерігачам) підписуватися на оновлення іншого об'єкта (суб'єкта) і отримувати повідомлення при зміні його стану.

У цьому прикладі:
- `WeatherStation` — суб'єкт, який зберігає поточну температуру.
- `ConsoleDisplay` — простий спостерігач, що виводить значення в консоль.
- `Alarm` — спостерігач, який реагує на перевищення порогу.

Переваги патерна:
- Слабка зв'язка між суб'єктом і спостерігачами.
- Додавання/видалення спостерігачів під час виконання.

Запуск прикладу: `npm run build` (за потреби) і потім `node` на зкомпільованому файлі або запустити TypeScript безпосередньо за допомогою `ts-node`.
